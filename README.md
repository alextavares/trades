# Trades Backup

Backup dos robos, backtests, dashboard e testes locais de trading.

Arquivos de ambiente reais, logs, CSVs e estados de runtime nao devem ser versionados. Use os arquivos `.env.*.example` como modelo e preencha as chaves apenas no servidor/local de execucao.

## Validacao rapida

```powershell
python -m pytest
```
